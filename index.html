Welcome to the course webpage

This pages gives you some background on the course presenters and provides links to the course materials and example solutions to the exercises.

Course materials

The course materials are available as IPython notebook files, one for each session of the course. IPython notebooks allow you to interactively run the example code. If you install IPython you can then run the notebooks locally and experiment with the code.
